The dependencies for these Python scripts are `numpy` and `sagemaker`. To install and use in a self contained virtual environment,

```
python -m venv .venv
source .venv/bin/activate.sh

pip install numpy sagemaker
```

For information on each script, see the READMEs in the ``SMT_COMP_2020`` and ``Eager_Lazy`` folders, and comments at the top of each script.
